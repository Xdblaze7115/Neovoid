# Neovoid
A Game Engine built using DirectX 11

Neovoid is a simple game engine built with DirectX 11 and C++.

## Features
- DirectX 11 rendering
- ImGui for debug UI
- 3D model rendering
- Basic camera controls
- Input handling

## Requirements
- Windows 10 or later
- Visual Studio 2022
- DirectX 11 compatible GPU

## Building

### Method 1 Build from Source
1. Clone the repository
2. Open `Neovoid.sln` in Visual Studio
3. Build the solution
4. Run the executable

### Method 2 Download Pre-built Release
1. Go to the Releases tab on GitHub
2. Download the latest `.zip` file
3. Extract the files
4. Launch `Neovoid.exe`

## Project Structure
- Source - Engine source code
- Resources - Sample assets used for testing
- Vendor - Third party libraries

## Purpose
This is a learning project to understand game engine development and DirectX 11 programming however I might expand it in the future.
